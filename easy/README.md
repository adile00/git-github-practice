# Git & GitHub Practice – Backend Course

This repository contains practical exercises for **Lesson 2** of the Backend Course.  
The purpose of this project is to practice real Git and GitHub workflows used in software development.

Through these tasks, students learn how to work with commits, branches, stash operations, and remote repositories.

---

##UPDATE

## Easy Level

The **easy** level focuses on basic Git operations:

- Initializing a Git repository
- Creating files
- Staging changes
- Making commits
- Pushing code to GitHub

The goal is to understand the basic workflow used in almost every Git project.
